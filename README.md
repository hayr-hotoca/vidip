# Install environment

NodeJS version >= 20

Rust

Static ffmpeg executables: Download from https://ffmpeg.martin-riedl.de/ then put it in the app `src-tauri/binaries` folder

# Install dependencies

Open cmd/terminal, navigate to the root folder then run:

`yarn`

# Run the app

Open cmd/terminal, navigate to the root folder then run:

`yarn tauri dev`
